🩺 XGBoost Machine Learning Classification Project using Diabetes Dataset

📌 Project Overview
This project uses the XGBoost Classification algorithm to predict whether a patient has diabetes based on medical attributes. The objective is to build an accurate machine learning model and evaluate its performance.
🎯 Problem Statement
To predict whether a person is diabetic or non-diabetic using healthcare data and machine learning techniques.

📂 Dataset Information
The dataset contains medical records with the following features:

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age
- Outcome (Target Variable)
